🔬 Die Akademie der jungen Forscher

Wissenschafts-Geburtstag für Jonte, Emil, Mia & Fiete – 4 Kinder · ca. 2 Stunden

🧪 Die Geschichte

Dr. Neutrinos „Superformel" ist bei einem missglückten Experiment in Fragmente zerfallen. Die vier Kinder werden als Nachwuchs-Forscher in die Akademie aufgenommen – mit Kittel, Ausweis und Forscher-Eid. An den Stationen sammeln sie Ziffern, knacken damit den Code am Zahlenschloss der Forscher-Truhe und lösen zum Schluss den großen Effekt aus.

Der Geheimcode: Vier Stationen liefern je eine Ziffer, das Detektivlabor verrät den Fundort der Truhe. Der Code lautet 2 0 1 7
(Station 1 → 2, Station 2 → 0, Station 3 → 1, Station 5 → 7). Stell diesen Code am Zahlenschloss der Truhe ein.

🔭 Die fünf Stationen

Chemie

Station 1 – Der Geheimtrank

Rotkohlsaft als natürlicher pH-Indikator. Die Kinder tröpfeln Zitronensäure (→ pink/rot) und Natronlösung (→ blau/grün) in Becher und mischen die „richtige Trankfarbe".

🎯 Signal: Wer die Zielfarbe trifft, erhält die Ziffer 2.

Material: Rotkohl/Rotkohlsaft, Natron, Zitronensäure, klare Becher, Pipetten.

Chemie · Wärme

Station 2 – Die Geheimbotschaft

Mit Zitronensaft vorgeschriebene Botschaft wird mit dem Föhn sichtbar (sicherer als Bügeleisen).

🎯 Signal: Die enthüllte Nachricht nennt die Ziffer 0.

Material: Zitronensaft, weißes Papier, Wattestäbchen, Föhn.

Physik · E-Technik

Station 3 – Der Stromkreis-Test

Batteriebetriebene Messstation mit LED/Summer und zwei Kontakten. Die Kinder testen Gegenstände auf Leitfähigkeit – Löffel leitet, Radiergummi nicht.

🎯 Signal: Beim leitenden Objekt leuchtet die LED → dahinter klebt die Ziffer 1.

Material: AA-Batteriehalter, LED (o. 3-V-Summer), 220-Ω-Widerstand, Litze, Krokoklemmen. Gehäuse aus dem 3D-Drucker (siehe unten).

⚠️ Nur Batteriespannung – nie Netzspannung.

Biologie

Station 4 – Das Detektivlabor

Fingerabdrücke nehmen (Bleistift-Abrieb, mit Klebefilm abheben) und Muster vergleichen: Schleife, Wirbel, Bogen. „Wessen Abdruck war an der Formel?"

🎯 Signal: Der Hinweis verrät den Fundort der Truhe.

Material: weiche Bleistifte, Klebefilm, weißes Papier, Lupen.

Physik

Station 5 – Der Dichte-Turm

Schichten unterschiedlicher Dichte in ein hohes Glas: Honig → Spülmittel → gefärbtes Wasser → Öl. Sie bleiben getrennt und bilden einen Regenbogen-Turm.

🎯 Signal: In welcher Schicht ein Gegenstand liegen bleibt, gibt die Ziffer 7 frei.

Material: hohes Glas, Honig, Spülmittel, Speiseöl, Lebensmittelfarbe.

🌋 Finale – „Der große Ausbruch"

Elefantenzahnpasta light, aufgebaut in einer großen Wanne oder auf einem Backblech – funktioniert dadurch drinnen wie draußen: Trockenhefe in warmem Wasser aktivieren, in eine Flasche mit Spülmittel, Lebensmittelfarbe und 3 %igem Wasserstoffperoxid geben → dicker, warmer Schaum quillt heraus.

⚠️ Sicherheit: Strikt bei 3 % H₂O₂ aus der Drogerie/Apotheke bleiben. Höherprozentiges (>12 %) ist reizend/ätzend und für Kindergeburtstage unnötig. Schutzbrillen auf – dann ist der Effekt harmlos und spektakulär.

Outdoor-Bonus (nur bei gutem Wetter): zusätzlich Filmdöschen-Raketen mit Brausetablette. Als Backup einfach weglassbar.

🥼 Laborkittel selber basteln

Günstig & sauber aus einem übergroßen weißen T-Shirt (oder alten weißen Hemd):

  • T-Shirt flach hinlegen, Vorderseite oben.
  • Vorne mittig senkrecht aufschneiden – vom Halsausschnitt bis zum Saum. Das ist die Kittelöffnung.
  • Ärmel bei den Kleinen kürzen/umschlagen.
  • Schnittkanten mit Textilkleber oder ein paar Stichen sichern.
  • Mit Textilstift/Bügelfolie ein „Akademie"-Logo + Namensfeld auf die Brust malen; ein aufgemalter Kugelschreiber in der Brusttasche macht was her.

Aus altem Hemd noch einfacher: ist vorne schon offen – nur Logo drauf. ~10 Min pro Kittel.

Kaufalternative (D): „Kinder Laborkittel / Arztkittel Kostüm" bei Amazon.de (~9–14 €), saisonal auch buttinette.de.

🖨️ 3D-Druck – OpenSCAD

1) Forscher-Messstation (Gehäuse + Deckel)

// Forscher-Messstation – parametrisch (mm)
box_l = 100; box_w = 70; box_h = 40;
wall  = 2.4;  lid_h = 3;
screw_d = 4.2;        // Bohrung M4-Kontaktschrauben
screw_spacing = 45;   // Abstand der Kontakte
led_d = 5.2;          // 5-mm-LED  (Summer: 12.5)
$fn = 48;

module gehaeuse() {
  difference() {
    cube([box_l, box_w, box_h]);
    translate([wall, wall, wall])
      cube([box_l-2*wall, box_w-2*wall, box_h]);
  }
}
module deckel() {
  difference() {
    cube([box_l, box_w, lid_h]);
    translate([box_l/2 - screw_spacing/2, box_w/2, -1])
      cylinder(d=screw_d, h=lid_h+2);
    translate([box_l/2 + screw_spacing/2, box_w/2, -1])
      cylinder(d=screw_d, h=lid_h+2);
    translate([box_l/2, box_w/2 + 20, -1])
      cylinder(d=led_d, h=lid_h+2);
  }
}
gehaeuse();
translate([0, box_w + 15, 0]) deckel();

2) Flaschenhalter fürs Finale

// Kippsicherer Flaschenhalter (mm)
bottle_d = 65;   // Flaschendurchmesser messen!
ring_h  = 40;  base_d = 130;  wall = 4;
$fn = 64;
difference() {
  union() {
    cylinder(d=base_d, h=6);
    translate([0,0,6]) cylinder(d=bottle_d+2*wall, h=ring_h);
  }
  translate([0,0,6]) cylinder(d=bottle_d, h=ring_h+1);
}
